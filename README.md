# Tripleten web_project_around (Around The U.S.)

An interactive web project showcasing modern web development techniques, responsive design, and modular code organization using HTML, CSS, and JavaScript (ES Modules).

## Features

-   **Modular CSS:** Organized styles in the `blocks/` directory (e.g., [`header.css`](blocks/header.css), [`popup.css`](blocks/popup.css))
-   **Responsive Design:** Media queries ensure a smooth experience on both desktop and mobile devices.
-   **Dynamic Interactivity:** Profile editing and popup functionalities implemented in [`popup.js`](scripts/popup.js) and [`submitProfileForm.js`](scripts/submitProfileForm.js).
-   **Code Quality:** Configured with [Prettier (`.prettierrc`](.prettierrc))](.prettierrc) and [EditorConfig (`.editorconfig`](.editorconfig)) for consistent styling.

## Project Structure

-   **HTML:** Core structure defined in [`index.html`](index.html)
-   **CSS:** Component styles in `blocks/`, global styles in [`pages/index.css`](pages/index.css), and third-party styles in [`vendor/`](vendor/)
-   **JavaScript:** ES Modules in the `scripts/` folder with [`index.js`](scripts/index.js) as the entry point.
-   **Assets:** Organized images and fonts in the `images/` and [`vendor/fonts/fonts.css`](vendor/fonts/fonts.css) directories.

## Getting Started

1. Clone the repository.
2. Open [`index.html`](index.html) in your browser to see the project in action.
3. Use Visual Studio Code for an enhanced development experience.

## Technologies

-   HTML5 & CSS3
-   JavaScript (ES Modules)
-   Responsive design
-   Prettier, EditorConfig
